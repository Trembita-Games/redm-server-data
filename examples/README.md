# Examples

Example configuration snippets for using resources from `redm-server-data`.

---

## Current Examples

- [server.cfg](server.cfg) — example resource include section for an existing RedM/RDR2 server configuration

---

## Purpose

The files in this directory are examples only.

They are not intended to replace the full server configuration from:

```txt
redm-vanilla-template
```

Use them as reference snippets when adding standalone resources from this repository to your own server.

## Recommended Base Server

Recommended base repository:

```txt
https://github.com/Trembita-Games/redm-vanilla-template
```

## Notes

This repository does not replace default Cfx.re resources.

Resources from `redm-server-data` should be added on top of an already working RedM/RDR2 server setup.
