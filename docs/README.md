## Current Focus

The initial focus is to investigate and document world-related issues seen on minimal RedM/RDR2 servers, including:

- missing buildings
- missing glass
- missing collisions
- partially loaded locations
- walk-through structures
- multiplayer validation issues

---

## Scope

This repository should remain focused on standalone resources and server data improvements.

It should not become a full RP framework or a large monolithic resource pack.
