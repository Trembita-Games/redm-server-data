---

## Resource Requirements

Each resource added to this repository should include:

- clear purpose
- installation instructions
- required dependencies
- `server.cfg` example
- known limitations
- testing notes

---

## Compatibility Rules

Resources should be:

- standalone where possible
- framework-agnostic
- compatible with `redm-vanilla-template`
- documented before being recommended
- separated from RP systems unless explicitly scoped otherwise

Resources should not require:

- VORP
- RSG
- inventory systems
- economy systems
- jobs systems
- character systems
